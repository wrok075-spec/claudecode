name: No confirmation questions
description: User wants autonomous execution — never ask yes/no or "want me to do X?" Just do it.
type: feedback

Don't ask for confirmation or yes/no questions. Just execute.

**Why:** User finds it slows down the workflow. He knows what he wants and expects Claude to act immediately.

**How to apply:** When the analysis recommends an action (pause ads, upload creatives, change settings), do it immediately in the same response. Don't end with "Want me to do X?" — just do X. Only ask questions when genuinely missing information (like a COGS number or API key).
