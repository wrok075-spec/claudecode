# Claude Code Configuration

All custom Claude Code skills, commands, and configuration for DTC eCommerce agency work.

## Structure

```
commands/           # Slash commands (invoke with /command-name)
skills/             # Skills with SKILL.md + references
project-memory/     # Memory files for the Kiwinz project
stores/             # Shopify store aliases
settings.json       # Claude Code settings
project-skills-SKILL.md  # Static ads project skill definition
```
